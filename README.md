## AI (Artificial Intelligence)
Making machines perform tasks that require human intelligence.
Examples: Chatbots, face recognition, Google Maps.

## ML (Machine Learning)
A subset of AI where machines learn from data without explicit rules.
Examples: Spam filtering, recommendations, fraud detection.

## DL (Deep Learning)
A subset of ML using deep neural networks for complex data.
Examples: Speech recognition, image recognition.
